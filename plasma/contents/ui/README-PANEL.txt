Para usar este Plasmoid como widget horizontal de painel no KDE Plasma:

1. Adicione-o diretamente ao painel como widget/appwidget.
2. Não o coloque dentro da área de bandeja do sistema (system tray), pois ali o Plasma o tratará como um ícone pequeno.
3. Após adicionar ao painel, redimensione a largura do widget no modo de edição do painel.

Se ele for colocado na bandeja, o Plasma vai comprimir a representação compacta para o tamanho de ícone e o layout horizontal não será respeitado.
